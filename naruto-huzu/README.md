# naruto-huzu — Frontend (React + Vite)

Este subprojeto contém um cliente React configurado com Vite para desenvolvimento rápido. Use-o como ponto de partida para o front-end do jogo.

## Rodando localmente

Pré-requisitos: Node.js 18+.

```bash
cd naruto-huzu
npm install
npm run dev
```

O dev server abre a aplicação em `http://localhost:5173` (ou porta indicada pelo Vite).

## Build para produção

```bash
npm run build
npm run preview
```

## Observações

- Ajuste as chamadas à API no código-fonte para apontar para a instância correta da API (`http://localhost:5000` por padrão).
- Recomendamos adicionar lint e CI conforme o padrão do repositório principal.
