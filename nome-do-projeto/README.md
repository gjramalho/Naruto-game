# nome-do-projeto — Frontend (React + Vite)

Este diretório contém um cliente React usado no projeto. As instruções abaixo ajudam a rodar e empacotar a aplicação.

## Rodando em desenvolvimento

Pré-requisitos: Node.js 18+.

```bash
cd nome-do-projeto
npm install
npm run dev
```

O servidor abre a aplicação em `http://localhost:5173` (Vite pode escolher outra porta se a 5173 estiver em uso).

## Build e preview

```bash
npm run build
npm run preview
```
